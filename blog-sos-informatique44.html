Veille technologique automatique

La Veille IA scanne les flux RSS des plus grands sites tech (ZDNet, 01net, Frandroid...), recupere les articles du jour, puis les reecrit avec Claude en version SEO locale optimisee pour SOS Informatique 44.

Les articles reecrits contiennent vos mots-cles locaux (La Chapelle-sur-Erdre, Nantes, Loire-Atlantique), les balises H1/H2/H3, et un appel a l'action vers votre site.

Selectionnez vos sujets a gauche puis cliquez sur Chercher des articles.

Cle API Claude (Anthropic)

Votre cle API est utilisee pour la generation d'articles en mode Pro et la reecriture IA dans la veille. Elle est stockee uniquement dans votre navigateur (sessionStorage).

Protection de votre cle API La cle est stockee de facon protegee (encodee) dans votre navigateur. Elle n'est JAMAIS envoyee ailleurs qu'a l'API Anthropic officielle. Pour une securite maximale, utilisez plutot un proxy HTTPS (section ci-dessous) qui garde la cle cote serveur. Vous pouvez supprimer la cle a tout moment.

Modele IA

Choisissez le modele Claude pour la generation et la reecriture d'articles. Haiku est recommande : 4x moins cher que Sonnet, qualite largement suffisante pour du SEO local.

Proxy API (Mode Pro Blog)

Si vous ne souhaitez pas exposer votre cle API directement, configurez un proxy HTTPS qui gere la cle cote serveur.
